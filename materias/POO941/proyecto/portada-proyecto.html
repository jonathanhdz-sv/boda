DESCRIPCION DEL PROYECTO

Sistema web para la administracion de la Biblioteca del Colegio Amigos De Don Bosco. Permite gestionar usuarios, libros y prestamos. Incluye catalogo publico sin login, modulo de administracion para encargados y modulo de prestamos para alumnos y profesores con calculo automatico de mora.

Antes: cuadernos, papeles perdidos, no se sabe quien tiene cada libro. Ahora: todo digital, desde cualquier lugar, en segundos.
